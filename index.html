File Requirements:

  • Maximum 500,000 rows
  • Must include 'latitude' and 'longitude' columns
  • Comma-separated CSV format only (not semicolon or tab-separated)
  • Coordinates must be numeric: latitude (-90 to 90), longitude (-180 to 180)
  • Example: latitude,longitude
    40.7128,-74.0060
    51.5074,-0.1278
  • If using fire data from FIRMS, favor using the VIIRS satellite data
  • Dates should be in the format YYYY-MM-DD. Other formats still work, but may have issues with filtering.
